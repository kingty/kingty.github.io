为什么我们需要重写这两个方法？

那为什么需要重写hashcode()呢?

hashcode()是用于散列数据的快速存取，如利用HashSet/HashMap/Hashtable类来存储数据时，都是根据存储对象的hashcode值来进行判断是否相同的.
